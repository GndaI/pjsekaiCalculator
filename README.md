# pjsekaiCalculator
Calculator of Project Sekai in Arcaea's potential calculation method.

-Calculation

Perfect:1

Great:0.5

Good:0.25

Bad:0.125

Miss:0

-Grade

AP:10000000

EX+:9900000 ~ 9999999

EX:9800000 ~ 9899999

AA:9500000 ~ 9799999

A:9200000 ~ 9499999

B:8900000 ~ 9199999

C:8600000 ~ 8899999

D:0 ~ 8599999

-Potential Calculation

9800000 or more: Difficulty+(Score-9800000)/200000-1

Under 9800000: Difficulty+(Score-9500000)/300000-2
